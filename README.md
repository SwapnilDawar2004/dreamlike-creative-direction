# dreamlike — creative direction

A single-page creative-direction and visual moodboard site for **dreamlike2004** — an atmospheric web experience built around the idea that *your life is performance art*.

**Live:** https://dreamlike-creative-direction.vercel.app

## About

This is a designed, self-contained web page (no framework, no build step) that presents the visual world and creative direction of the dreamlike2004 project — hero, film-grain texture, transition effects, and an archive gallery of visuals.

## Run locally

Open `index.html` in any modern browser, or serve it:

```bash
python3 -m http.server
# then open http://localhost:8000
```

## Structure

| Path | What it is |
|------|------------|
| `index.html` | The full page — layout, styles, and interactions |
| `instagram-dreamlike2004-*/` | Source images used in the visual archive |
